Au clic sur le lien dans le cadre blanc, un cookie "acceptsCookies" sera crée. Il aura une durée de vie de 1 journée.

Le cookie aura pour valeur true.

Si le cookie est défini et qu'il vaut true, alors le message dans le cadre blanc ne doit pas apparaître.

En revanche, vous devrez faire apparaitre un deuxième cadre blanc avec un bouton "Supprimer le cookie". Au clic de ce bouton, le deuxième cadre blanc et le cookie doivent être supprimés, et le message doit réapparaitre.

Ce site utilise les cookies, cliquez sur OK si vous acceptez leur utilisation. OK
