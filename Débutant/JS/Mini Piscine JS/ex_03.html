3

Faire en sorte de faire apparaitre une fenêtre de dialogue affichant "Quel est votre nom ?" au clic sur le cadre blanc.

Si aucun nom n'est renseigné, la fenêtre doit continuer à s'afficher jusqu'à ce qu'un nom soit renseigné.

Une fois un nom renseigné, faire apparaître une fenêtre de confirmation affichant "Etes vous sûr que *nom* est votre nom ?"

Si le nom est confirmé, une alerte avec le contenu "Bonjour *nom* !" doit s'afficher. Ce contenu sera également affiché dans le cadre blanc.
